# ContentPro 🚀

Landing page profesional para **ContentPro** — servicio de contenido para redes sociales.

## Estructura del proyecto

```
contentpro/
├── index.html        # Página principal
├── css/
│   └── styles.css    # Estilos
├── js/
│   └── main.js       # Animaciones e interacciones
└── README.md
```

## Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/contentpro.git
   ```
2. Abre `index.html` en tu navegador.

## Despliegue en GitHub Pages

1. Ve a **Settings** → **Pages**
2. En *Source* selecciona `main` branch
3. ¡Tu sitio estará live en `https://tu-usuario.github.io/contentpro`!

## Tecnologías

- HTML5
- CSS3 (variables, animaciones, grid)
- JavaScript vanilla (IntersectionObserver, contadores)
- Google Fonts: Syne + DM Sans
